퀴즈 생성기 사용 안내

이 퀴즈 생성기는 일괄 입력 ↔ 개별 편집이 좌우로 연동되어,
문제 / 답1 / 답2 / 답3 형식의 텍스트를 입력하면 자동으로 테이블에 반영됩니다.
복수정답 처리는 답과 답 사이에 ,를 넣어주면 됩니다.

어디에 쓸 수 있나요?

  • 학습 플랫폼: 교사용 자료 제작, 학생 셀프 테스트
  • 스터디 그룹: 토론 주제 배포 전 간단한 선테스트
  • 온라인 강의: 슬라이드에 바로 붙여넣어 실시간 퀴즈
  • 개인 복습: 출퇴근길·여행 중 모바일 학습
  • 자료 정리 및 개념 요약: 노트 대용이나 학습 컨텐츠 정리 도구로 활용

어떻게 사용하나요?

  1. 문제 편집 화면에서 문제 텍스트 / 답1 / 답2 / 답3 를 엔터로 구분하여 입력하세요.
    복수정답은 ,로 구분합니다.
    빈 칸은 자동으로 제거되며 최대 3칸까지 지원합니다.
  2. 오른쪽 테이블에서 개별 행을 직접 수정할 수도 있습니다. 수정 즉시 왼쪽 입력창에 동기화됩니다.
  3. “문제 풀기” 버튼을 누르면 퀴즈 모드로 전환됩니다.
  4. 퀴즈 화면에서는 제출, 전체 재풀이, 틀린 문제만 재풀이 기능을 제공합니다. 오답 시에는 올바른 답을 함께 확인할 수 있습니다.

* 이 퀴즈 생성기는 족보 풀이 이외 모든 과목·주제에 활용 가능하며, 학습 내용을 체계적으로 정리·요약하는 툴로도 사용할 수 있습니다.
